Enable Debug Mode

If you need to get a better look into what Fabric is doing behind the scenes, feel free to enable debug mode:

// Objective-C
[[Fabric sharedSDK] setDebug: YES];

[Fabric with:@[CrashlyticsKit]];

// Swift
Fabric.sharedSDK().debug = true
Fabric.with([Crashlytics.self()])

And this is how it looks in Ruby

def upload_large_icon!(icon_path)
  unless icon_path
    @large_app_icon.reset!
    return
  end
  upload_image = UploadFile.from_path icon_path
  image_data = client.upload_large_icon(self, upload_image)

  @large_app_icon.reset!({ asset_token: image_data['token'], original_file_name: upload_image.file_name })
end

For example, you can get insight into whether Answers is enabled and sending events.
